使用说明 - 水平无缝拼接

🚀 推荐：批量上传方式

  1. 准备2-8张樊振东传奇历程的图片文件（建议尺寸：16:9比例）
  2. 将图片文件按时间顺序重命名，确保文件名排序正确
  3. 点击"选择图片"，一次性选择所需数量的图片
  4. 点击"批量处理"按钮，系统自动按顺序加载并调整布局

📁 备选：单个上传方式

  1. 首先选择要拼接的图片数量（2-8张）
  2. 按照时间顺序分别上传图片到对应的输入框
  3. 系统将根据选择的数量自动调整拼接布局

🎯 生成和使用

  1. 点击"生成拼接图片"按钮创建水平无缝拼接图
  2. 生成的图片将是7张图片从左到右完全无缝连接的单行布局
  3. 预览效果满意后，点击"下载拼接图片"保存到本地
  4. 将下载的图片重命名为 hero-collage.jpg
  5. 将图片放置到项目的 public/images/carousel/ 目录下
  6. 刷新网站查看水平拼接的开场轮播图效果

技术规格：输出图片为1920×1080像素，支持2-8列动态布局，等比例缩放保留完整原图，完全无缝拼接。

等比例缩放：采用智能缩放算法，完整保留每张图片内容，不进行任何裁剪，只进行等比例放大或缩小。

动态适应：根据图片数量自动调整每张图片宽度，2张图片各占1/2，3张各占1/3，以此类推到8张。

完整保留：所有图片内容100%保留，空白区域自动填充深色背景，确保视觉协调。

质量提升：95%JPEG质量输出，轻微对比度增强，确保最佳显示效果。
